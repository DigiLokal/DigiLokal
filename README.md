## Getting Started

### Prerequisites

- Android OS Phone (Lollipop/API 21+)

or

- Android Studio (with built-in emulator)

### Installation

#### Android Studio

1. Clone the repo
   ```sh
   git clone https://github.com/DigiLokal/DigiLokal
   ```
2. Build the apps
   ```sh
   Run `app`
   ```
